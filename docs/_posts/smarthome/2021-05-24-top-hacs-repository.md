### Мой личный список используемых репозиториев HACS и не только.

Управление инфракрасным пультом-шайбой Broadlink mini.

[smartHomeHub/SmartIR](https://github.com/smartHomeHub/SmartIR/)

Управление устройствами Sonoff без перепрошивки.

[AlexxIT/SonoffLAN](https://github.com/AlexxIT/SonoffLAN)

Симпатичная карточка с барометром

[SNoof85/lovelace-tempometer-gauge-card](https://github.com/SNoof85/lovelace-tempometer-gauge-card)

Масштабная прокачка кнопок

[custom-cards/button-card](https://github.com/custom-cards/button-card)

Подстрочные значения и подписи

[custom-cards/secondaryinfo-entity-row](https://github.com/custom-cards/secondaryinfo-entity-row)

Шикарная карта квартиры для пылесосов Xiaomi

[PiotrMachowski/lovelace-xiaomi-vacuum-map-card](https://github.com/PiotrMachowski/lovelace-xiaomi-vacuum-map-card)

Минималистичная карточка для управления пылесосом

[benct/lovelace-xiaomi-vacuum-card](https://github.com/benct/lovelace-xiaomi-vacuum-card)

Удобная карточка для пылесоса с возможностью задать зоны для уборки на кнопки

[denysdovhan/vacuum-card](https://github.com/denysdovhan/vacuum-card)

Погодная карточка с анимированными иконками

[bramkragten/weather-card](https://github.com/bramkragten/weather-card)

Карточка для отображения качества воздуха с Air Visual

[dnguyen800/air-visual-card](https://github.com/dnguyen800/air-visual-card)

Сворачивающиеся карточки

[RossMcMillan92/lovelace-collapsable-cards](https://github.com/RossMcMillan92/lovelace-collapsable-cards)

Уведомлятор и счетчик дней для разных событий и юбилеев

[mf-social/ps-date-countdown](https://github.com/mf-social/ps-date-countdown)
